# onishi50-slide

A **Markdown slide tool** built with *Ratatui* + tachyonFX.

Press `→` to go to next slide.

---

## Features

- Parse Markdown and render in terminal
- Scroll with `j`/`k` keys
- **Bold**, *Italic*, ~~Strikethrough~~
- `inline code` support
- Page navigation with `←`/`→`
- Per-slide layouts
